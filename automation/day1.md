# Automatisierung (Bash)

## Ziel
Erste Grundlagen der Automatisierung in Linux mit einem einfachen Bash-Skript.

## Durchgeführte Schritte
- Erstellung eines Bash-Skripts (sysinfo.sh)
- Nutzung grundlegender Systembefehle im Skript:
  - whoami
  - hostname
  - uptime
  - df -h
  - free -h
- Ausführbar machen des Skripts mit chmod +x
- Ausführung des Skripts im Terminal

## Ergebnis
Ein einfaches Skript wurde erstellt, das grundlegende Systeminformationen automatisch anzeigt.

## Wichtige Erkenntnisse
- Bash-Skripte ermöglichen die Automatisierung von Linux-Befehlen
- Mehrere Befehle können in einer Datei zusammengefasst werden
- Wiederkehrende Systemprüfungen können automatisiert werden
