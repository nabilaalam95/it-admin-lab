# Systemsteuerung

## Verwendete Befehle

* ps aux → laufende Prozesse anzeigen
* top → Systemauslastung in Echtzeit anzeigen
* ps aux | grep bash → nach einem Prozess suchen
* sleep 1000 & → Testprozess starten
* ps aux | grep sleep → Testprozess finden
* kill PID → Prozess beenden
* whoami → aktuellen Benutzer anzeigen
* hostname → Systemnamen anzeigen
* uptime → Laufzeit und Systemlast anzeigen

## Wichtige Erkenntnisse

* Prozesse können überwacht und beendet werden.
* Jeder Prozess besitzt eine eindeutige PID.
* Linux bietet Werkzeuge zur Überwachung von Systemressourcen.
* Systeminformationen können direkt über das Terminal abgerufen werden.
