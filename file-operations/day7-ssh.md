# Tag 7 – SSH Remote-Zugriff

## Ziel
Simulation der Remote-Administration eines Servers mit SSH.

## Durchgeführte Aufgaben
- OpenSSH-Server installiert und konfiguriert  
- Status des SSH-Dienstes überprüft  
- IP-Adresse des Servers ermittelt  
- Lokalen SSH-Zugriff (localhost) getestet  
- Grundlegende Administrationsbefehle remote ausgeführt  
- Funktionsweise von SSH für sicheren Zugriff verstanden  

## Wichtige Erkenntnisse
- SSH ermöglicht sicheren Remote-Zugriff auf Server  
- Systeme können ohne direkten Zugriff administriert werden  
- Lokaler SSH-Zugriff simuliert reale Remote-Umgebungen
