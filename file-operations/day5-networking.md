# Tag 5 – Netzwerkgrundlagen

## Verwendete Befehle
- ip a → IP-Adresse und Netzwerkinterfaces anzeigen  
- ping google.com → Internetverbindung testen  
- hostname → Systemnamen anzeigen  
- ss -tuln → offene Ports und Dienste anzeigen  

## Wichtige Erkenntnisse
- Der Netzwerkstatus kann über das Terminal überprüft werden  
- Die IP-Adresse identifiziert ein System im Netzwerk  
- Ping dient zur Überprüfung der Internetverbindung  
- Offene Ports zeigen aktive Dienste auf dem System
